# 🌦️ Weather Forecast App

A responsive and user-friendly weather forecast application built using HTML, CSS, and JavaScript. The application fetches real-time weather data from the OpenWeather API and displays current weather conditions along with a 3-day forecast.

## ✨ Features

- 🌍 Search weather information by city name
- 📍 Automatic location detection using device GPS
- 🌡️ Real-time temperature in Celsius
- ☁️ Dynamic weather conditions and icons
- 💧 Current humidity information
- 💨 Wind speed details
- 🌡️ Feels-like temperature
- 📅 Automatic date and time updates
- 🔮 3-day weather forecast
- 📱 Fully responsive design
- 💻 Works on mobile, tablet, laptop, and desktop
- 🔒 HTTPS support through GitHub Pages
- ⚠️ Error handling for invalid city searches

## ⚙️ How It Works

1. The user opens the weather application.
2. The application can request the user's current location through GPS.
3. The user can also search for any city manually.
4. JavaScript sends a request to the OpenWeather API.
5. Current weather information is displayed dynamically.
6. Forecast data is processed and displayed for the upcoming days.
7. Date and time are updated automatically.

## 🚀 How to Run Locally

1. Clone this repository:

git clone https://github.com/Mahieesai/WeatherForcastApp.git

2. Open the project folder.

3. Add your OpenWeather API key inside `script.js`.

4. Open `index.html` using a browser or VS Code Live Server.

## 🔑 API Configuration

This project uses the OpenWeather API.

Create an API key from the OpenWeather platform and add it to your JavaScript configuration.

Example:

const apiKey = "YOUR_API_KEY";

> Important: Never publish a private or sensitive API key in a public repository.
